Overview

Sudoku Puzzle Solver is a simple Eclipse view that allows you to solve Sudoku puzzles.  You can create a new blank puzzle, enter pre-set values yourself, and then solve the puzzle. Or you can have a puzzle with pre-set values generated for you.

To start the Sudoku Puzzle Solver, that is to open the Sudoku view, select the Sudoku menu item and then Open Sudoku Puzzle.
